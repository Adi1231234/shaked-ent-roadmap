# שקד - מפת דרכים לרפואת אף-אוזן-גרון

אתר אישי שמרכז 28 תוכניות, קורסים והזדמנויות לבניית רזומה לקראת התמחות ברפואת אא״ג, ממוינים לפי ROI (יוקרה מול זמן השקעה).

## פיתוח מקומי

```bash
npm install
npm run dev
```

## בנייה

```bash
npm run build
```

## פריסה

פריסה אוטומטית ל-GitHub Pages דרך GitHub Actions כאשר נדחפים שינויים ל-`main`.
